Security policy

If you discover a security vulnerability, please report it privately to the repository maintainers.

Contact: vishverwasma (via GitHub) or open a private security advisory.

Policy:
- Do not post security issues in public issues if they are sensitive
- Provide steps to reproduce and any PoC files
- Maintainers will respond within 48 hours

Notes on code execution:
- The project disables remote code execution by default (POLYMATH_ALLOW_CODE_EXECUTION env var).
- To run examples locally, enable only on trusted hosts and avoid running in CI.
