Polymath AI Assistant — Resume blurb

Developed "Polymath AI", a modular command-line assistant that integrates with LLMs (OpenAI) to perform advanced tasks from a single prompt: question answering, document summarization (PDF/TXT), and CSV data analysis. Built with a clean, testable architecture (unit tests, CI), the project demonstrates engineering skills in API integration, CLI design, data processing, and automated testing.

Highlights:
- Implemented a pluggable LLM wrapper and resilient fallbacks for local testing
- Built document extraction and summarization pipeline for PDFs
- Added CSV analysis utilities with optional Pandas support
- Automated verification: tests run repeatedly to demonstrate stability
